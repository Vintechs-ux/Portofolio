Node.js

Apa itu Node.js?

Node.js adalah runtime JavaScript yang dibangun di atas mesin V8 JavaScript Chrome. Ini memungkinkan pengembang untuk menjalankan kode JavaScript di server, membuka kemungkinan untuk pengembangan full-stack dengan satu bahasa pemrograman.

  • Asynchronous dan Event-driven
  • Single-threaded dengan event looping
  • Cross-platform dan open source
  • Performa tinggi untuk aplikasi real-time

Use Cases

Web Applications

Membangun aplikasi web yang scalable dan real-time

API Development

Membuat REST API dan microservices

Real-time Apps

Chat applications dan game servers

Command Line Tools

Build tools dan automation scripts

Key Features

Non-blocking I/O

Operasi I/O asynchronous yang meningkatkan throughput dan efisiensi

NPM (Node Package Manager)

Akses ke lebih dari 1 juta package dan modul

Getting Started

$ npm init
$ npm install express
$ node app.js
